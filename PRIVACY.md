# Privacy — Guardians of Galia

Guardians of Galia is a browser-only game. It does not require an account, name,
email address, chat profile, or file upload. Game progress and settings are stored
locally in the player’s browser.

Clearing browser storage, using private browsing, or changing devices can remove
progress. Players can export a save backup from Settings and import it later.

The current public build has no behavioral analytics, advertising trackers, or
in-game feedback form. Children should not send personal information when asking
an adult or the person who shared the game for help.

Last updated: 24 July 2026.
